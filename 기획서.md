# Daily ToDoList 기획서

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | Daily ToDoList |
| 목적 | 노션 칸반 보드 형태의 일정 생성·관리 웹 서비스 |
| 타겟 | 개인 업무·일정 관리가 필요한 누구나 |
| 플랫폼 | 웹 (PC 중심, 모바일 반응형 지원) |
| 기술 스택 | HTML5 / CSS3 / Vanilla JS / LocalStorage |

---

## 2. 핵심 기능 정의

### 2-1. 칸반 보드

| 컬럼 | 설명 |
|------|------|
| 할 일 | 아직 시작하지 않은 태스크 |
| 진행 중 | 현재 진행 중인 태스크 |
| 완료 | 완료된 태스크 |

- 컬럼별 태스크 카드 목록 표시
- 컬럼별 태스크 수 뱃지 표시

### 2-2. 태스크 카드

카드에 포함되는 정보:

| 필드 | 타입 | 필수 여부 |
|------|------|----------|
| 제목 | 텍스트 | 필수 |
| 설명 | 텍스트 | 선택 |
| 마감일 | 날짜 | 선택 |
| 우선순위 | 높음 / 보통 / 낮음 | 선택 (기본: 보통) |

### 2-3. 태스크 CRUD

| 기능 | 설명 |
|------|------|
| 생성 | 각 컬럼 하단 `+ 카드 추가` 버튼 → 모달 팝업 |
| 조회 | 칸반 보드에서 카드 목록 확인 |
| 수정 | 카드 클릭 → 수정 모달 팝업 |
| 삭제 | 카드 내 삭제 버튼 클릭 |

### 2-4. 드래그 앤 드롭

- 카드를 다른 컬럼으로 드래그하여 상태 변경
- 같은 컬럼 내 순서 변경 가능

### 2-5. 데이터 영속성

- LocalStorage에 자동 저장
- 새로고침 후에도 데이터 유지

## 3. UI 구조

```
┌─────────────────────────────────────────────────────┐
│  Daily ToDoList                     오늘 날짜 표시   │
├──────────────┬──────────────┬───────────────────────┤
│   할 일  [n] │  진행 중 [n] │      완료  [n]        │
│              │              │                       │
│ ┌──────────┐ │ ┌──────────┐ │  ┌──────────┐        │
│ │ 카드 제목 │ │ │ 카드 제목 │ │  │ 카드 제목 │       │
│ │ 설명...  │ │ │ 설명...  │ │  │ 설명...  │        │
│ │ 📅 날짜  │ │ │ 📅 날짜  │ │  │ 📅 날짜  │        │
│ │ 🔴 높음  │ │ │ 🟡 보통  │ │  │ 🟢 낮음  │        │
│ └──────────┘ │ └──────────┘ │  └──────────┘        │
│              │              │                       │
│ + 카드 추가  │ + 카드 추가  │   + 카드 추가         │
└──────────────┴──────────────┴───────────────────────┘
```

### 모달 구조 (생성/수정 공통)

```
┌──────────────────────────────┐
│  태스크 추가 / 수정           │
│  ─────────────────────────── │
│  제목 *  [________________]  │
│  설명    [________________]  │
│          [________________]  │
│  마감일  [날짜 선택        ]  │
│  우선순위 ● 높음 ○ 보통 ○ 낮음│
│                              │
│     [취소]       [저장]      │
└──────────────────────────────┘
```

## 4. 디자인 가이드

| 항목 | 값 |
|------|-----|
| 주 색상 | `#5B6CF6` (인디고) |
| 배경 | `#F7F8FC` |
| 카드 배경 | `#FFFFFF` |
| 폰트 | Pretendard / 시스템 폰트 fallback |
| 우선순위 높음 | `#FF5C5C` 빨강 |
| 우선순위 보통 | `#FFA500` 주황 |
| 우선순위 낮음 | `#4CAF50` 초록 |
